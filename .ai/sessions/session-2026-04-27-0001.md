### 2. SSE 중간 이벤트 스트리밍 (높음)
현재 `_stream_run()`은 `AgentLoop.run()` 완료 후 결과만 SSE로 감싸는 구조.
apt-web UI가 기대하는 중간 이벤트(`orchestrator.role.invoke.*`, `role.tool.call.*`,
`orchestrator.todo.change`, `orchestrator.critic.verdict`, `input_required`)가 전달되지 않음.

필요 작업:
- `AgentLoop`에 event callback/queue 메커니즘 추가
- `_stream_run()`에서 queue를 소비하여 SSE frame 발행

### 3. HITL interrupt-resume 완성 (높음)
`/a2a/respond` 구조는 존재하나 `ask_user` 콜백 ↔ asyncio.Future ↔ SSE 스트림 연동 미완.
`AgentLoop.run(ask_user=callback)` 에서 callback이 `_pending_interrupts`에 Future 등록 →
SSE로 `input_required` emit → `/a2a/respond`가 Future resolve → 그래프 재개.

### 4. apt-web 통합 (`.ai/apt-web-integration-v1-instructions.md`)
작업지시서 존재하나 §8 반영 필요:
- A2A 응답 형식 변경 (dummy → 실 envelope)
- SSE event 이름 매핑 (`task.start/artifact/status/end` vs `orchestrator.*`)
- `AGENT_CODING_BASE_URL` 설정

### 5. E2E 검증
apt-web UI → ax-coding-agent 실제 코드 생성 작업 완주 테스트.
